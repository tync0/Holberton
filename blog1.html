Installing Mango Automation on CentOS

Mango SCADA is an open-source, web-based SCADA (Supervisory Control and Data Acquisition) system that allows users to monitor and control industrial processes.

Prerequisites

Before you begin, make sure you have the following:

  • A CentOS machine with root access
  • Java Runtime Environment (JRE) 8 or newer
  • At least 1GB of RAM

Step 1: Install Java

Mango SCADA requires Java to run. You can install the latest version of Java by running the following command:

sudo yum install java-1.8.0-openjdk

Step 2: Download Mango SCADA

You can download the latest version of Mango SCADA from the official website. Choose the appropriate package for your machine's operating system.

curl -o install-mango.sh https://raw.githubusercontent.com/MangoAutomation/ma-core-public/main/Core/bin/install-mango.sh
chmod +x install-mango.sh
sudo ./install-mango.sh

Step 3: Run the script

The script will prompt you to enter details about where to install Mango, which version of Mango to install, etc. Hit enter and accept the default option shown inside the square brackets [default]. Yes/no answers must be written as ‘yes’ or ‘no,’ ‘y’ or ‘n’ are not sufficient.

In my case, I use all defaults.

sh install-mango.sh

Step 4: Start Mango

After the installation was successful, we need to start our local host. Write the following command:

systemctl start mango

You can check the status of the Mango:

systemctl status mango

Mango should now be running on your machine. You can access the Mango web interface by visiting http://localhost:8443/ in your web browser.

Step 5: Log in

For the default username is admin, and the password is admin.

{"username": "admin","password": "admin"}

Push agree with the term after login.

Conclusion

You have successfully installed Mango SCADA on your CentOS machine. You can now start monitoring and controlling your industrial processes with Mango.
